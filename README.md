# file_organizer

A lightweight command-line utility to automatically sort cluttered directories by moving files into categorized subfolders based on their extensions.

## Features
* Automatically categorizes Images, Documents, Videos, and Audio files.
* Fully customizable extension mapping via `config.json`.
* Detailed execution logging.
